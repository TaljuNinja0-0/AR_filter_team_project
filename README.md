# AR Filter

PySide6, OpenCV, Dlib 기반의 AR 필터 데스크톱 앱입니다. 이미지와 동영상을 불러와 얼굴 인식 결과를 바탕으로 다양한 필터를 적용할 수 있습니다.

## 주요 기능

- 이미지와 동영상 파일 불러오기
- 필터 메뉴에서 원하는 AR 효과 선택
- 선택한 필터를 화면에 즉시 적용
- 이미지 저장 또는 동영상 전체 저장
- 얼굴 랜드마크 기반 필터와 반응형 필터 지원

## 지원 필터

앱 메인 화면에서 총 18개의 필터를 선택할 수 있습니다.

- 왜곡, 글리치, 카메라, 빈티지
- 안경, 모노클, 강아지, 고양이, 토끼
- 무대 가면, 콧수염, 흰 마스크, 하트, 선글라스, 마스크
- 불, 시선 트래킹, 잔상

## 동작 방식

- [AR_app_main.py](AR_app_main.py)에서 UI를 로드하고 필터 버튼과 미디어 입력을 연결합니다.
- 각 필터는 별도 모듈의 `apply_*` 함수로 연결되어 있습니다.
- 이미지 모드에서는 선택한 필터를 즉시 반영하고, 동영상 모드에서는 프레임마다 필터를 적용합니다.
- 필터 메뉴를 닫으면 현재 선택된 필터 기준으로 저장 작업이 실행됩니다.

## 실행 조건

이 프로젝트는 얼굴 랜드마크 검출을 위해 `shape_predictor_68_face_landmarks.dat` 파일이 필요합니다. 프로젝트 루트에 파일을 두어야 정상 동작합니다.

반응형 필터 중 일부는 특정 표정이나 동작이 있는 영상에서만 자연스럽게 동작합니다. 예를 들어 하트 필터는 눈 감기 또는 윙크, 불 필터는 입 벌리기 같은 반응이 있어야 효과가 잘 보입니다.

## 설치

가상환경을 사용하는 경우 먼저 활성화한 뒤, 의존성을 설치합니다.

```bash
pip install -r requirements.txt
```

## 실행

```bash
python AR_app_main.py
```

## 사용 방법

1. 앱을 실행한 뒤 미디어 파일을 불러옵니다.
2. 메뉴에서 필터 목록을 열고 원하는 필터를 선택합니다.
3. 이미지면 즉시 미리보기가 갱신되고, 동영상이면 재생 프레임에 계속 적용됩니다.
4. 필터 메뉴를 닫으면 현재 상태를 기준으로 저장 작업이 진행됩니다.

## 저장 결과

- 이미지 저장: 원본 파일명 뒤에 필터명이 붙은 새 이미지로 저장됩니다.
- 동영상 저장: 원본 파일명 뒤에 필터명이 붙은 새 mp4 파일로 저장됩니다.

## 프로젝트 구조

- [AR_app_main.py](AR_app_main.py): 앱 진입점 및 UI/필터 연결
- [ui_AR_Filter.py](ui_AR_Filter.py): Qt Designer로 생성된 UI 코드
- `*_filter.py`, `*_filter_mapping.py`: 개별 필터 구현
- `assets/`, `assets_moving/`, `icons/`: 필터 이미지와 아이콘 리소스

## 참고 사항

- 일부 반응형 필터는 영상 입력에서 가장 잘 동작합니다.
- 새 필터를 추가할 때는 [AR_app_main.py](AR_app_main.py)에서 `filter_map`, `filter_names`, `filter_icons`, `filter_media_support`를 함께 갱신해야 합니다.
- 각 필터 함수는 `frame` 하나를 입력받아 `frame`을 반환하는 형태여야 합니다.
